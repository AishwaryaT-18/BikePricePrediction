# Bike Price Prediction 🚲

## Overview
This project predicts the selling price of used bikes based on various features such as year, kilometers driven, fuel type, seller type, and more. It uses a machine learning model trained on historical bike data.

---

## Features
- Predicts bike selling price instantly
- Simple web interface for input
- Trained ML model for accurate predictions
- User-friendly UI with HTML/CSS

---

## Tech Stack
- Python
- Machine Learning(Random Forest Regressor)
- Pandas, NumPy

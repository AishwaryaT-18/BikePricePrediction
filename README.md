# Bike Price Prediction 🚲

## Overview
This project predicts the selling price of used bikes using machine learning techniques. The model is trained on historical bike data and evaluated using standard regression metrics.

---

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA) with graphs
- Feature selection and transformation
- Price prediction using Machine Learning model
- Performance evaluation using metrics

---

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn (for visualization)
- Scikit-learn (Random Forest Regressor)
- Jupyter Notebook

---

## Project Workflow
1. Data loading and cleaning  
2. Exploratory Data Analysis (EDA) using graphs  
3. Feature engineering  
4. Model training (Random Forest Regressor)  
5. Model evaluation using MAE, MSE, RMSE  

---

## Model Details
- Algorithm: Random Forest Regressor
- Metrics Used:
  - MAE (Mean Absolute Error)
  - MSE (Mean Squared Error)
  - RMSE (Root Mean Squared Error)

---

## How to Run
1. Open Jupyter Notebook
2. Run all cells step by step
3. View graphs and prediction results
